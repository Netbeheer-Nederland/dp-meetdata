    Meetdata Dataproduct

    The Meetdata data product describes the metered data for a single or multiple market evaluation points (Aansluiting). The Solution Design provides more detail on the use cases and implementation of the data product for DSOs. This data product follows the EUMED Profile, as standardised in ISO/IEC 61968-9:2024: Interfaces for meter reading and control. This data product aligns with the implementation by other EU Member States.

    The data is intended to be represented as a hierarchy with a MeterReadingDataSet at its root, followed by MarketEvaluationPoint-s nested inside the MeterReadingDataSet. Each marketEvaluationPoint has one or more Meter-s, where a Meter can have an associated Customer and one or more MeterReading-s.

    A MeterReading has one or more IntervalBlock-s, which defines the time period for which the IntervalReading-s are applicable. A ReadingType defines attributes for the IntervalReading-s. Each IntervalReading can be provided a ReadingQuality, defining data quality for the IntervalReading.
